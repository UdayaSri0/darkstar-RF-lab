Corrected wiring summary

Assembly order

  1. Build and test the regulated 5 V input, common ground and dedicated 3.3 V RF rail.
  2. Connect the main ESP32, one nRF24L01 module and its local decoupling capacitors.
  3. Add the remaining RF receivers one at a time, keeping every unused CS/CSN line high.
  4. Add the display and MicroSD bus, then the I²C expander and sensors.
  5. Add the ESP32-C5 as a dual-band Wi-Fi survey coprocessor.
  6. Add the filtered AD8318 detector and calibrate its ADC readings with known reference levels.
  7. Only after the basic system is stable, add the optional Linux SDR host branch.
