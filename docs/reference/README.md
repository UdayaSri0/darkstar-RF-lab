# Legacy reference material

This directory preserves the original component research documents supplied
with the repository:

- `rf-signal-analyser-components.pdf`
- `rf-signal-analyser-components.docx`

These files are retained for provenance and are **not the authoritative
DarkStar RF Lab specification**. They pre-date the current receive-only scope
and may contain earlier concepts, conflicting pin assignments, unsupported
claims, or component choices that have not been validated.

Use the following current project documents instead:

- [`../../README.md`](../../README.md) for scope and project status;
- [`../hardware/wiring-reference.md`](../hardware/wiring-reference.md) for the
  corrected controller wiring allocation; and
- [`../../web-guide/index.html`](../../web-guide/index.html) for the interactive
  circuit guide.

Any future design decision taken from the legacy material must be checked
against current manufacturer documentation and reviewed before implementation.
